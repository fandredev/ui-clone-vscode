## Clone this repo

```
git clone git@github.com:fandredev/ui-clone-vscode.git
```

## Go to directory

```
 cd ui-clone-vscode
```

## Install dependencies

```
 yarn install
```

## Run server

```
 yarn run dev
```

## Run tests

```
 yarn run test
```

## Run coverage

```
 yarn run coverage
```
